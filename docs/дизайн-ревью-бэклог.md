## ⏳ Точечные отложенные технические находки

- [ ] **`barFill` `transition: width` → `transform: scaleX`** (perf, impeccable `[layout-transition]`). Невидимо глазу, ~8 мест (Reports, Clients, Directive, Dashboard) + `styles.js`. Низкий приоритет.
- [ ] **Загрузочные состояния заменяют весь экран** спиннером (видно в Register/Directive: `if (loading) return <spinner>`). Лучше держать крутилку только в зоне контента, не теряя hero/фильтры. Проверить по экранам.
