# ADR-0005. Демонтаж технической интеграции ManaJet

- **Дата**: 2026-07-01 · **Статус**: принято (решение заказчика) · PR #218
- **Решение**: зеркальные таблицы `mj_*`, ночной синк (cron + Edge Functions),
  панели «Наши данные / ManaJet» удалены. ManaJet остаётся **функциональным
  ориентиром** (доменная модель): `docs/manajet-логика.md` и `docs/manajet-gap-map.md`
  живут как конспект и роадмап.
- **Почему**: обкатка зеркала завершена, свой финконтур покрывает потребности;
  двойной контур данных создавал путаницу и расходы на синхронизацию.
- **Что меняет**: миграция `20260701090000_drop_manajet_mirror.sql`; `mj_secret` →
  `app_secret`; импортированные ранее справочники (funds/income_types/…) — рабочие
  данные, НЕ тронуты.
