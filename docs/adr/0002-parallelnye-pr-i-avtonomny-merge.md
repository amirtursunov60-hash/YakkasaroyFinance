# ADR-0002. Параллельные PR и автономный мерж

- **Дата**: 2026-06-25 · **Статус**: принято (решение заказчика)
- **Решение**: (а) правило «один PR за раз» отменено — параллельные PR разрешены;
  (б) Claude мержит самостоятельно, без команды заказчика, после ревью-инструментов
  и зелёных `typecheck`/`test`/`lint`/`build` + CI. Исключение: значимый риск или
  неоднозначность из ревью — сначала вопрос через AskUserQuestion.
- **Почему**: скорость итераций; заказчик смотрит результат по preview-ссылкам.
- **Что меняет**: раздел «Рабочий процесс» CLAUDE.md; в чат заказчику — только
  короткая preview-ссылка Vercel (не ссылки на PR/дашборд).
