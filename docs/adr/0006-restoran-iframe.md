# ADR-0006. Модуль «Ресторан» — iframe приложения yakkasaroy-menu

- **Дата**: 2026-06-30 · **Статус**: принято · PR #212
- **Решение**: в продукте «Ресторан» — это iframe отдельного приложения
  `yakkasaroy-menu` (репозиторий `yakkasaroy-Pos-and-menu`, свой деплой Vercel);
  тема/язык синхронизируются postMessage-протоколом (`yk-finance` ↔ `yk-restaurant`).
  Нативные мок-экраны `Rest*` в Finance — только дизайн-референс.
- **Почему**: публичное меню и POS живут своим релизным циклом; по ТЗ v2 §4.11
  операционную часть в перспективе заменит интеграция iiko.
- **Что меняет**: сайдбар Finance (один пункт «Ресторан»), правило в CLAUDE.md обоих
  репозиториев; дизайн-система Ресторана — копия темы Finance (сверка
  `scripts/check-theme-sync.sh` там).
