# ADR-0008. Осознанно вне периметра продукта

- **Дата**: 2026-06-24 · **Статус**: принято (решение заказчика)
- **Решение**: НЕ делаем в обозримом периметре: налоговый контур (НДС и отчётность),
  складской учёт (остатки/себестоимость — придёт с интеграцией iiko), прямую
  банковскую интеграцию (выписки руками через Контроль средств).
- **Почему**: фокус на финансовом планировании по ХМС; перечисленное — отдельные
  продукты с высокой ценой поддержки, часть закроется iiko.
- **Что меняет**: раздел «Осознанно вне периметра» в `docs/manajet-gap-map.md` —
  первоисточник; новые запросы фич сверять с этим списком, расширение — только
  новым ADR.
