# ADR-0004. Tailwind v4 + shadcn для нового, инлайн-стили ядра остаются

- **Дата**: июнь 2026 · **Статус**: принято (решение владельца)
- **Решение**: сосуществование, не «или-или». Новые компоненты/экраны можно строить
  на Tailwind v4 (без preflight, поддеревья в `tw-scope`) и shadcn; существующие
  экраны на инлайн-стилях из `src/theme/` не переписываются ради миграции (Boy Scout).
  Цвета — только палитра `C`; Tailwind-токены ссылаются на CSS-переменные `--c-*`.
- **Почему**: не останавливать фичи ради тотальной миграции; глобальный reset ломал бы
  готовые экраны.
- **Что меняет**: раздел «Стиль» CLAUDE.md; отменяет «без Tailwind» из ранних документов
  (README поправлен, UI-бриф помечен).
